## 各页面功能说明

### 首页 `/`
- 顶部导航栏（Logo、搜索框、登录/头像、发帖按钮、AI助手入口）
- Banner区（热门话题 / 公告）
- 「为你推荐」（AI个性化推送，登录后显示）
- 「最新帖子」列表（标题、摘要、作者、板块标签、时间、点赞数）
- 「热帖榜」侧边栏（Top 10）
- 「活跃板块」侧边栏
- 底部（关于、联系、规则）

### 分区总览页 `/section`
- 4大板块卡片（图标、名称、简介、今日发帖数）
- 点击进入对应分区页

### 分区页 `/section/:name`
- 面包屑导航
- 子分类标签切换
- 帖子列表（支持排序：最新/最热/已解决）
- 发帖按钮（快速发到当前分区）
- 分区规则侧边栏

### 详细分区页 `/section/:name/:sub`
- 同上，固定到某一子分类
- 筛选条件（时间范围、是否有图、是否已解决）

### 帖子详情页 `/post/:id`
- 帖子标题、AI生成摘要、标签
- 正文（富文本）
- 作者信息卡（头像、学校、等级）
- 点赞、收藏、分享、举报操作栏
- 「AI为你总结」按钮（一键总结长帖）
- 评论区（楼层显示、支持引用回复）
- 相关推荐帖子（AI推送）

### 发帖页 `/post/create`
- 选择板块 / 子分类
- 标题输入
- 富文本编辑器（支持图片、代码块、表格）
- 「AI生成摘要」按钮（实时调用）
- 「AI生成标签」按钮
- AI内容质量检测提示
- 发布 / 存草稿

### 个人主页 `/user/:id`
- 头像、昵称、学校（已认证显示✓）、等级、简介
- 统计（发帖数、获赞数、粉丝数）
- 发帖列表 / 收藏列表 Tab
- 关注 / 私信按钮

### 搜索结果页 `/search`
- 搜索框（关键词高亮）
- 筛选：全部 / 帖子 / 用户 / 板块
- 按相关度 / 时间 / 热度排序

### AI智能助手页 `/ai-assistant`
- 对话界面（类ChatGPT）
- 自动引用论坛内相关帖子
- 历史对话记录

### 后台首页 `/admin`
- 今日数据卡片（新增用户、发帖数、AI调用次数）
- 待处理事项（待审核帖子数、未处理举报数）
- 折线图（7日活跃趋势）
